Course Description

This course is an advanced graduate course that will introduce students to the core literacies required of technical and professional communicators. In this course, students will learn and apply concepts related to information literacy, source literacy, and technical communication concept literacy. Students will be required to create documents using source code including HTML/CSS for content display and XML for data storage. Students will also be required to research and apply technical procedures and concepts on their own. This course is designed for students who have a desire to learn source code. Although you don't have to be familiar with source code, you must WANT to learn it to succeed in this course.
